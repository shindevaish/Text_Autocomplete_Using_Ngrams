# Text Autocomplete Using N-Gram Model (NLP)

## 📌 Project Overview

This project demonstrates the implementation of **N-Gram Language Models** for **text prediction and sentence auto-completion** using Natural Language Processing (NLP) techniques.

The project uses **Bigram, Trigram, and 4-gram models** to analyze text data and predict the next word in a sentence. It also includes **word cloud visualization** and **prediction comparison** between different N-gram models.

---

## 🎯 Project Objectives

- Apply **N-Gram models** in NLP
- Generate **Bigram, Trigram, and 4-gram sequences**
- Build **Word Prediction System**
- Develop **Sentence Auto-Completion System**
- Compare **Bigram vs Trigram performance**
- Visualize results using **WordCloud**

---

## 📊 Dataset Used

### 1. IMDB Movie Review Dataset
- Used for N-gram analysis
- Generated bigrams and trigrams
- Built prediction system

### 2. News Articles Dataset (Kaggle)
- Generated 4-gram sequences
- Built auto-completion system
- Compared prediction accuracy
